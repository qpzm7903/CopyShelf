# CopyShelf

Windows 桌面端的片段启动器（snippet launcher）：用户预先配置常用文本片段，通过全局快捷键呼出搜索框，选中后粘贴到前台窗口。**不是**剪贴板历史管理器——不捕获、不记录系统剪贴板的复制历史，永远不做。

## Language

**CopyShelf**:
本产品。一个 Windows 片段启动器。
_Avoid_: 剪贴板管理器、剪贴板历史工具

**片段 (Snippet)**:
用户预先配置的一段纯文本，被选中后粘贴到前台窗口。只被粘贴，永不执行。
_Avoid_: 指令、命令、Command

**目标窗口 (Target Window)**:
按下呼出快捷键那一瞬间的前台窗口，片段最终粘贴到这里。若它在选中片段前已关闭，粘贴降级为仅复制到剪贴板。
_Avoid_: 前台窗口（呼出后前台窗口是 CopyShelf 自己，二者必须区分）

**片段定义 (Snippet Definition)**:
片段中由用户编辑、跨设备同步的部分：名称、内容、描述、标签。

**使用统计 (Usage Stats)**:
片段在本台设备上的使用频率与最近使用时间。仅本地记录，不跨设备同步（见 ADR-0001），只用于排序。
_Avoid_: 频率数据、热度

**关键词匹配 (Keyword Match)**:
搜索词对片段名称、描述、标签的子串匹配，中文字段额外支持全拼与首字母拼音命中。不是 fuzzy 子序列匹配。
_Avoid_: 模糊匹配、fuzzy（除非真的实现了子序列匹配）
