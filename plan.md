# CopyShelf 产品与版本计划

> 本文件只记录产品方向、版本范围和交付状态。维护流程、质量门禁和发布步骤见
> [docs/maintenance.md](docs/maintenance.md)，竞品依据见
> [docs/product/competitive-analysis-2026-07.md](docs/product/competitive-analysis-2026-07.md)。

## 产品目标

把 CopyShelf 做成 Windows 上最快、最可信、对开发者和重度文本用户最友好的主动片段启动器。

产品边界保持稳定：本地优先、纯文本、用户主动呼出、片段只粘贴不执行；不记录剪贴板历史，不默认监听全部键盘输入，不把 Git 同步变成托管账号服务。

## 当前基线

- 最新已发布版本：`v0.1.25`。
- 当前候选版本：`v0.2.0`，开发中。
- 自动化基线：`300` 项测试通过，`2` 项平台条件测试跳过。
- 发布目标：Windows 10 及以上，GitHub Actions 产出 x86_64 ZIP。
- 当前外部阻塞：无开放 GitHub Issue；无可见的失败 CI 运行。

## 版本计划

### v0.2.0 (MINOR) - 可靠性与高频路径

**目标**：消除会挂住应用或破坏用户配置的故障，同时缩短“发现内容到保存片段”的路径。

- [x] `Ctrl+Enter` 仅复制，不切换目标窗口。
- [x] 搜索窗 `Ctrl+N` / 新建按钮快速创建，搜索词预填名称、剪贴板预填内容。
- [x] Git 命令使用非交互模式、超时和进程终止。
- [x] 快捷键重绑成功后才持久化，失败恢复旧组合。
- [x] 自动化回归测试。
- [ ] Windows 实机验收：全局快捷键、窗口失焦、剪贴板预填、Git 凭据失败。（剩余风险：发布自 macOS 开发机，实机验收待发布后在 Windows 上补做）
- [x] 更新版本号、发布说明并完成 Release 验证。

### v0.2.1 (PATCH) - 可靠性回合

**目标**：在下一个功能版本前，集中处理 `v0.2.0` 的用户反馈、CI 问题和结构债务。

- [ ] 修复已验证的 `v0.2.0` 回归。
- [ ] 复查快速创建、仅复制、Git 超时和快捷键恢复的错误路径。
- [ ] 清理剩余静态分析 info，建立可执行的格式化基线。
- [ ] 完成 Windows 10/11、DPI、多显示器、RDP 验收矩阵。

### v0.3.0 (MINOR) - 可恢复与可迁移

**目标**：让用户敢于长期积累片段，并能随时迁移数据。

- [ ] 回收站与单条恢复，删除不再依赖手动查 Git 历史。
- [ ] 稳定的 JSON / CSV 导出，导出格式带 schema 版本。
- [ ] CSV 与至少一种主流文本扩展器导入。
- [ ] 导入预览、重复项处理和失败报告。
- [ ] 隐私安全的本地诊断日志与一键导出；严禁记录片段正文、剪贴板内容和凭据。
- [ ] 建立恢复、迁移和日志脱敏测试。

### v0.3.1 (PATCH) - 可靠性回合

**目标**：在下一个功能版本前，集中处理 `v0.3.0` 的用户反馈、CI 问题和结构债务。

- [ ] 修复已验证的 `v0.3.0` 回归。
- [ ] 检查导入、导出和恢复路径的数据完整性。
- [ ] 收紧慢测试和共享状态测试，降低偶发失败。
- [ ] 完成 Windows 10/11、DPI、多显示器、RDP 验收矩阵。

### v0.4.0 (MINOR) - 模板可用性

**目标**：增强模板表达力，但不引入脚本执行能力。

- [ ] 变量插入菜单与即时预览。
- [ ] `{cursor}` 光标位置。
- [ ] 日期格式、日期偏移和常用文本变换。
- [ ] 选项型字段（下拉/列表，对标 Espanso choice/list）、日期选择器和可选段落。
- [ ] 多行文本字段。
- [ ] 片段引用与循环依赖检测（对标 Raycast：嵌套限一层）。

### v0.5.0 (MINOR) - 上下文与生态

**目标**：把"呼出时已知目标窗口"变成差异化能力，并降低获取与分享片段的摩擦。依据见竞品分析"2026-07 深度调研增量"。

- [ ] 窗口上下文感知：按目标进程名/窗口标题自动预选标签或加权排序（复用 TargetWindowService 与标签体系，不做键盘监听）。
- [ ] 便携版 ZIP 与 Chocolatey 分发（winget 维持不做）。
- [ ] 片段包导入：从他人片段 Git 仓库 URL 一键导入，带预览与去重。
- [ ] 探索模糊搜索模式，先验证与拼音索引的融合成本。

## 探索项

自动缩写只有在以下前置条件全部满足后才进入正式版本：默认关闭、逐条启用、密码和凭据窗口禁用、不持久化按键、支持应用排除列表，并通过 Windows 实机兼容矩阵。AI、富文本、图片片段和脚本执行当前不进入路线图。

## 成功指标

- 呼出到完成粘贴的 P50 / P95 时长。
- 首次输入命中率、零结果率和搜索改词次数。
- 粘贴失败率、目标窗口丢失率和同步失败率。
- 新建一条可复用片段所需时间。
- 发布后 14 天内的崩溃、回归和紧急补丁数量。
