# 29: 标签栏布局重设计（常驻 + 过滤胶囊）

## Parent

issue 28 的设计迭代。用户对照竞品截图（备忘快贴）反馈：左侧应像截图一样把标签展示出来。

## Design decisions

- **标签栏常驻**：库非空即显示（issue 28 是"有标签才显示"）。没有标签时显示引导文案
  「编辑片段时可添加标签」——空态即邀请，让用户发现标签这个组织方式。空库仍是单栏空态。
- **`#` 前缀替代通用 label 图标**：标签项渲染为 `# tag`，与已有 `#tag` 搜索语法互相呼应
  （产品自己的词汇，而非 Material 默认图标）。固定项（全部/无标签）保留小图标。
- **搜索栏过滤胶囊**：选中标签后搜索栏内出现 `# tag ×` 胶囊（对照截图顶栏的标签 chip），
  点 × 清除；**空输入框按 Backspace 也清除**——标签输入框的通用肌肉记忆，键盘党零成本退出过滤。
- 固定项与标签列表间发丝线分组；标签多时列表独立滚动。
- 「无标签」仅在混合库（有标签也有无标签片段）显示——全库无标签时它与「全部」等价。
- 悬停态（theme hoverColor）+ 手型光标；选中态仍是 accentTint 底 + accent 文字，不引入新颜色。
