# 05: 窗口生命周期 + 系统托盘

## Parent

`.scratch/copyshelf-v1/PRD.md`

## What to build

把 CopyShelf 变成真正的常驻型工具，窗口行为遵循启动器惯例（grilling 会话决议，PRD「窗口生命周期」一节）：

- **失焦即隐藏**：搜索窗口失去焦点（用户点击别处 / Alt+Tab 离开）时自动隐藏；删除现有的失焦抢回焦点逻辑；
- **关闭即托盘**：主窗口点 X 隐藏到托盘，进程不退出；
- **托盘图标与菜单**：常驻托盘图标，右键菜单至少含「打开设置」「退出」；退出是结束进程的唯一入口；
- **快捷键 toggle**：窗口可见时按 `Ctrl+Alt+V` 隐藏窗口（保持现有行为并联调验证）。

含 plan.md 中遗留的「系统托盘集成调试」。

## Acceptance criteria

- [ ] 呼出搜索窗口后点击其他应用，CopyShelf 窗口自动隐藏
- [ ] 主窗口点 X 后进程仍在（托盘图标可见），再按快捷键可正常呼出
- [ ] 托盘右键菜单「退出」彻底结束进程，快捷键注册被清理（退出后按快捷键无反应）
- [ ] 托盘右键菜单「打开设置」能打开设置页
- [ ] 窗口可见时按快捷键隐藏、不可见时按快捷键呼出，往复多次无异常
